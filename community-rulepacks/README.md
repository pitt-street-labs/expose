# Community Rule Packs

This directory contains community-contributed rule packs for the EXPOSE
attribution engine. Each subdirectory is a self-contained pack with a
`rules.json` (the rule pack) and a `README.md` (documentation).

For contribution guidelines, pack structure, naming conventions, quality
requirements, and the review process, see
[docs/community-rulepacks.md](../docs/community-rulepacks.md).

For the rule pack schema, see
[schemas/rulepack-v1.json](../schemas/rulepack-v1.json).

For example packs shipped with EXPOSE, see
[examples/rulepacks/](../examples/rulepacks/).
